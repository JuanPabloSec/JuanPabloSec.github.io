# PrawdaNews - Prank News Website

## Opis
Strona internetowa typu "fake news" z artykułem prankowym o szkoleniu psów do wykrywania narkotyków.
**UWAGA:** Wszystkie informacje w artykule są fikcją!

## Pliki
- `index.html` - Strona główna
- `article.html` - Artykuł prankowy
- `style.css` - Stylizacja
- `script.js` - Skrypty

## Jak uruchomić lokalnie
Otwórz `index.html` w przeglądarce.

## Jak wgrać na GitHub
1. Utwórz nowe repo na GitHub (np. `PrawdaNews`)
2. Przeciągnij i upuść wszystkie pliki
3. Gotowe!
